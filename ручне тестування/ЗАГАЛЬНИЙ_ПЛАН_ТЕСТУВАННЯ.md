## Середовище тестування

### Вимоги

- **Node.js:** v18+
- **Go:** v1.21+
- **Браузери:** Chrome, Firefox, Safari, Edge
- **Мобільні:** iOS Safari, Android Chrome

### Запуск проекту

```bash
# Backend (Go)
cd services/core
go run cmd/server/main.go

# Frontend (Next.js)
cd services/storefront
npm run dev
```

### URL-адреси

| Середовище | URL |
|------------|-----|
| Development | http://localhost:3000 |
| Admin Panel | http://localhost:3000/admin |
| API | http://localhost:8080/api |

## Типи тестування

### 1. Функціональне тестування
- Перевірка бізнес-логіки
- Валідація форм
- CRUD операції
- Пошук та фільтрація

### 2. UI/UX тестування
- Responsive дизайн (мобільний, планшет, десктоп)
- Доступність (accessibility)
- Консистентність інтерфейсу
- Анімації та переходи

### 3. Інтеграційне тестування
- API взаємодія
- Синхронізація з маркетплейсами
- Платіжні системи
- Email/SMS сповіщення

### 4. Негативне тестування
- Невалідні дані
- Граничні значення
- Помилки мережі
- Паралельні операції

### 5. Тестування продуктивності
- Час завантаження сторінок (< 3 сек)
- Час відповіді API (< 500 мс)
- Робота з великими обсягами даних
