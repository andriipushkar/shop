## Фаза 6: Маркетинг та лояльність

### 6.1 Промокоди та знижки

#### Створення промокоду
| # | Крок | Очікуваний результат | ✓/✗ |
|---|------|---------------------|-----|
| 1 | Перейти до Admin → Маркетинг → Промокоди | Відкривається сторінка промокодів | |
| 2 | Натиснути "Створити промокод" | Відкривається форма створення | |
| 3 | Заповнити код: TEST20 | Код відображається | |
| 4 | Вибрати тип знижки: Відсоток | Поле значення показує % | |
| 5 | Ввести значення: 20 | Значення прийнято | |
| 6 | Встановити мінімальну суму: 500 | Значення збережено | |
| 7 | Встановити дати дії | Дати відображаються | |
| 8 | Натиснути "Зберегти" | Промокод з'являється в списку | |

#### Застосування промокоду
| # | Крок | Очікуваний результат | ✓/✗ |
|---|------|---------------------|-----|
| 1 | Додати товари на суму > 500 грн | Товари в кошику | |
| 2 | Перейти до кошика | Відображається сума | |
| 3 | Ввести промокод TEST20 | Поле приймає код | |
| 4 | Натиснути "Застосувати" | Знижка 20% застосована | |
| 5 | Перевірити нову суму | Сума зменшилась на 20% | |
| 6 | Спробувати з сумою < 500 грн | Помилка мінімальної суми | |
| 7 | Ввести невірний код | Помилка "Код не знайдено" | |

#### Типи знижок
| Тип | Тест | Очікуваний результат | ✓/✗ |
|-----|------|---------------------|-----|
| Відсоток | 20% на 1000 грн | Знижка 200 грн | |
| Фіксована | 100 грн на 1000 грн | Знижка 100 грн | |
| Безкоштовна доставка | Застосувати код | Доставка 0 грн | |
| Купи X отримай Y | 3+1 | 4-й товар безкоштовно | |

## Чек-лист перед релізом

### Функціональність
- [ ] Всі промокоди працюють
- [ ] Бали нараховуються коректно
- [ ] Сертифікати активуються
- [ ] Email кампанії відправляються
- [ ] SEO мета-теги присутні
- [ ] Соціальні кнопки працюють
- [ ] Інвентаризація проводиться
- [ ] Постачальники керуються

### PWA
- [ ] Service Worker реєструється
- [ ] Офлайн режим працює
- [ ] Push сповіщення доставляються
- [ ] Встановлення на Android
- [ ] Встановлення на iOS

### Продуктивність
- [ ] Lighthouse Performance > 90
- [ ] First Contentful Paint < 2s
- [ ] Time to Interactive < 5s
- [ ] Bundle size оптимальний

### Безпека
- [ ] HTTPS на всіх сторінках
- [ ] CSP headers налаштовані
- [ ] Чутливі дані не в localStorage
- [ ] API endpoints захищені
