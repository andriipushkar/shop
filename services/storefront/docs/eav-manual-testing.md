# Ручне тестування EAV системи атрибутів

## Огляд

Цей документ описує процедури ручного тестування системи EAV (Entity-Attribute-Value) для атрибутів товарів в маркетплейсі.

---

## 1. Сторінка адміністрування атрибутів

### 1.1 Доступ до сторінки
- **URL**: `/admin/attributes`
- **Очікуваний результат**: Відображається сторінка управління атрибутами

### 1.2 Базове відображення
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Завантаження сторінки | Відкрити `/admin/attributes` | Відображається список атрибутів |
| Статистика | Перевірити картки статистики | Показує: Всього атрибутів, Фільтруючі, Груп, Опцій |
| Вкладки | Перевірити наявність вкладок | Є вкладки "Атрибути" та "Групи" |

### 1.3 Таблиця атрибутів
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Колонки таблиці | Переглянути заголовки | Є: Назва, Код, Тип, Дії |
| Бейджі типів | Переглянути бейджі в колонці "Тип" | Різні кольори для різних типів |
| Розгортання опцій | Клікнути на стрілку атрибута типу "Вибір" | Відображаються опції атрибута |

### 1.4 Пошук та фільтрація
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Пошук | Ввести "brand" в поле пошуку | Фільтруються результати за назвою |
| Фільтр по типу | Вибрати тип у випадаючому списку | Показуються тільки атрибути вибраного типу |

### 1.5 Додавання атрибута
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Відкриття модального вікна | Клікнути "Додати атрибут" | Відкривається форма |
| Поля форми | Перевірити поля | Є: Код, Назва, Тип |
| Закриття модального вікна | Клікнути "Скасувати" | Модальне вікно закривається |

### 1.6 Типи атрибутів
| Тип | Тест | Очікуваний результат |
|-----|------|---------------------|
| Текст (text) | Вибрати тип "Текст" | Базова форма |
| Число (number) | Вибрати тип "Число" | Додатково з'являється поле "Одиниці виміру" |
| Вибір (select) | Вибрати тип "Вибір" | З'являється секція "Опції" |
| Мультивибір (multiselect) | Вибрати тип "Мультивибір" | З'являється секція "Опції" |
| Колір (color) | Вибрати тип "Колір" | З'являється поле вибору кольору |
| Логіка (boolean) | Вибрати тип "Логіка" | Базова форма |
| Діапазон (range) | Вибрати тип "Діапазон" | Поля min/max |

## 2. Сторінка товару

### 2.1 Доступ до сторінки
- **URL**: `/product/[id]`
- **Очікуваний результат**: Відображається картка товару з атрибутами

### 2.2 Базове відображення
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Назва товару | Переглянути заголовок | Відображається назва |
| Бренд | Перевірити бренд | Відображається назва бренду |
| Ціна | Перевірити ціну | Відображається поточна ціна |
| SKU | Перевірити SKU | Відображається код товару |
| Рейтинг | Перевірити рейтинг | Відображаються зірки та кількість відгуків |
| Статус наявності | Перевірити наявність | Показано "В наявності" або "Немає в наявності" |

### 2.3 Варіанти товару
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Вибір кольору | Клікнути на кольоровий кружок | Вибір змінюється, оновлюється зображення |
| Вибір пам'яті | Клікнути на варіант пам'яті | Ціна оновлюється |
| Різниця в ціні | Переглянути варіанти | Показана різниця в ціні (+10 000 ₴) |

### 2.4 Характеристики товару
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Основні характеристики | Переглянути секцію | Показані ключові атрибути |
| Групи атрибутів | Переглянути заголовки груп | Є групи: Загальні, Дисплей, Продуктивність |
| Значення атрибутів | Перевірити значення | Відображаються з одиницями виміру |
| Логічні атрибути | Перевірити boolean | Показані з галочкою (✓) |
| Вкладки характеристик | Переключити на "Усі" | Показуються всі групи атрибутів |
| Розгортання груп | Клікнути на заголовок групи | Група розгортається/згортається |

### 2.5 Кошик
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Кількість | Змінити кількість (+/-) | Кількість оновлюється |
| Додавання в кошик | Клікнути "Додати в кошик" | Товар додано, з'являється повідомлення "Додано!" |

### 2.6 Навігація
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Breadcrumbs | Перевірити навігацію | Показано: Головна / Категорія / Товар |
| Кнопка "Назад" | Клікнути "Назад" | Повернення на попередню сторінку |
| Посилання на характеристики | Клікнути "Усі характеристики →" | Скрол до секції характеристик |

## 3. Сторінка категорії з фільтрами

### 3.1 Доступ до сторінки
- **URL**: `/category/[slug]` (наприклад, `/category/smartphones`)
- **Очікуваний результат**: Відображається список товарів з фільтрами

### 3.2 Базове відображення
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Заголовок категорії | Переглянути заголовок | Відображається назва категорії |
| Панель фільтрів | Переглянути ліву панель | Показано секцію "Фільтри" |
| Кількість товарів | Перевірити лічильник | Показано "Знайдено X товарів" |

### 3.3 Фільтр ціни
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Відображення | Перевірити секцію "Ціна" | Показано поля "Від" та "До" |
| Зміна діапазону | Ввести значення | Товари фільтруються |

### 3.4 Фільтр бренду
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Відображення | Клікнути на секцію "Бренд" | Показано чекбокси з брендами |
| Вибір бренду | Обрати один або декілька брендів | Товари фільтруються |

### 3.5 Динамічні фільтри (для електроніки)
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Діагональ екрану | Переглянути фільтр | Показані поля "Від" та "До" (дюйми) |
| Оперативна пам'ять | Клікнути на секцію | Показані чекбокси (4 GB, 8 GB, 16 GB) |
| Вбудована пам'ять | Переглянути фільтр | Показані варіанти (256 GB, 512 GB) |
| Колір | Клікнути на секцію "Колір" | Показані кольорові кружечки |
| Підтримка 5G | Переглянути логічний фільтр | Показано чекбокс "Так" |
| NFC | Переглянути логічний фільтр | Показано чекбокс |
| Бездротова зарядка | Переглянути логічний фільтр | Показано чекбокс |

### 3.6 Взаємодія з фільтрами
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Вибір RAM | Обрати "8 GB" | Чекбокс відмічено, товари фільтруються |
| Множинний вибір пам'яті | Обрати "256 GB" та "512 GB" | Обидва відмічені |
| Вибір кольору | Клікнути на кольоровий кружок | Колір обведено рамкою |
| Логічний фільтр | Обрати "Підтримка 5G" > "Так" | Товари фільтруються |

### 3.7 Фільтр кольорів
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Кольорові кружки | Переглянути секцію "Колір" | Показані кольорові кружки |
| Підказка | Навести на колір | Показана підказка з назвою та кількістю |
| Множинний вибір | Обрати кілька кольорів | Всі обрані відмічені рамкою |

### 3.8 Скидання фільтрів
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Кнопка "Скинути" | Обрати фільтри, клікнути "Скинути" | Всі фільтри очищені |
| Лічильник фільтрів | Обрати кілька фільтрів | Показано кількість активних фільтрів |

### 3.9 Мобільна версія
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Кнопка фільтрів | Зменшити вікно до мобільного розміру | Показана кнопка "Фільтри" |
| Модальне вікно | Клікнути на кнопку "Фільтри" | Відкривається модальне вікно з фільтрами |

### 3.10 Сортування
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Випадаючий список | Переглянути сортування | Показано "За популярністю" |
| Зміна сортування | Вибрати "Від дешевих до дорогих" | Товари пересортовуються |

### 3.11 Режим відображення
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Перемикач | Перевірити кнопки view | Є кнопки сітки та списку |
| Зміна виду | Переключити режим | Товари відображаються у новому вигляді |

## 5. Тестування продуктивності

### Метрики
| Метрика | Очікуване значення |
|---------|-------------------|
| LCP (Largest Contentful Paint) | < 2.5s |
| FID (First Input Delay) | < 100ms |
| CLS (Cumulative Layout Shift) | < 0.1 |
| Time to Interactive | < 3s |

### Тести навантаження фільтрів
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Швидке переключення фільтрів | Швидко клікати по різних фільтрах | UI не блокується |
| Велика кількість товарів | Категорія з 1000+ товарів | Фільтрація працює швидко |
| Множинні фільтри | Активувати 5+ фільтрів одночасно | Час відповіді < 1s |

## 6. Тестування доступності

### Клавіатурна навігація
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Tab навігація | Натискати Tab | Фокус переміщується по елементах |
| Enter/Space | Натиснути на фільтрі | Фільтр активується |
| Escape | Натиснути в модальному вікні | Модальне вікно закривається |

### Screen Reader
| Тест | Кроки | Очікуваний результат |
|------|-------|---------------------|
| Alt текст | Перевірити зображення | Є описовий alt текст |
| ARIA мітки | Перевірити форми | Є aria-label для елементів |
| Заголовки | Перевірити структуру | Правильна ієрархія h1-h6 |

## 7. Чек-лист для релізу

### Критичний функціонал
- [ ] Сторінка атрибутів завантажується без помилок
- [ ] Можна створити новий атрибут
- [ ] Сторінка товару відображає всі атрибути
- [ ] Варіанти товару працюють коректно
- [ ] Фільтри на сторінці категорії працюють
- [ ] Скидання фільтрів працює
- [ ] Мобільна версія фільтрів працює

### Регресійне тестування
- [ ] Існуючі тести проходять
- [ ] Немає консольних помилок
- [ ] Немає візуальних регресій
- [ ] API відповіді коректні
