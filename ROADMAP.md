# Дорожня карта

Плани розвитку Shop Platform.

## Візія

Створити найкращу open-source e-commerce платформу для українського ринку з підтримкою multi-tenancy, сучасними інтеграціями та відмінним Developer Experience.

## Пріоритети

Пріоритети визначаються на основі:

1. **Потреби користувачів** - feedback від спільноти
2. **Бізнес-цінність** - вплив на конверсію та дохід
3. **Технічний борг** - підтримка якості коду
4. **Безпека** - захист даних користувачів

## Версіонування

Проект дотримується [Semantic Versioning](https://semver.org/):

- **MAJOR** (x.0.0) - несумісні зміни API
- **MINOR** (0.x.0) - нова функціональність, зворотно сумісна
- **PATCH** (0.0.x) - виправлення помилок
